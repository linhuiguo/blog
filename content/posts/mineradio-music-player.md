# GitHub 热榜 8K 星的音乐播放器！免费开源，网易云 QQ 音乐都能听（小白版）

> 你是不是也烦了：想听歌，要么开会员，要么软件里全是广告？今天安利一个 GitHub 上 **8,000 多人收藏**的开源音乐播放器 **Mineradio**——电影镜头级的粒子视觉、3D 歌单架、网易云和 QQ 音乐都能接，**双击 exe 就能装，不用敲一行命令**。
> 本文从"在哪下载"讲起，**每一步都给小白看**。

## ⚠️ 先说清

1. **这软件完全免费开源**：Mineradio 是 GPL-3.0 开源协议，不收费、没广告。
2. **只支持 Windows 10 / 11**：Mac 和 Linux 暂不支持。
3. **需要已有网易云/QQ 音乐账号**：调用你的账号来播放。

---

## 〇、准备清单

- 一台 Windows 10 或 Windows 11 电脑
- 能联网
- 一个网易云音乐 / QQ 音乐账号（可选）

---

## 一、第一步：下载安装包

### 1.1 两个下载源

- **蓝奏云（国内快，推荐）**：https://xxhuber.lanzout.com/s/Mineradio
- **GitHub Release（备选）**：https://github.com/XxHuberrr/Mineradio/releases/tag/v1.1.1

> ⚠️ 只下载 `Mineradio-1.1.1-Setup.exe` 这一个文件！旁边那些 `Source code`、`.blockmap`、`win-unpacked` 都不要管。

### 1.2 蓝奏云下载

1. 打开上面蓝奏云网址。
2. 页面里有一个 exe 文件列表，点它。
3. 输入验证码（如果有的话）。
4. 点 **下载**，浏览器开始下 `Mineradio-1.1.1-Setup.exe`。
5. 去"下载文件夹"里找到它。

### 1.3 GitHub 下载（蓝奏云打不开时用）

1. 打开 GitHub Release 链接。
2. 往下翻找 **Assets** 区域。
3. 找 `Mineradio-1.1.1-Setup.exe`，点它下载。

---

## 二、第二步：安装（双击即可）

1. **双击**下载好的 `Mineradio-1.1.1-Setup.exe`。
2. 如果弹出 **Windows 已保护你的电脑**（SmartScreen）——这是正常安全提醒，不是病毒。点 **"更多信息"** → 再点 **"仍要运行"**。
3. 安装向导出来，一路点 **下一步**。
4. 装完后桌面出现 **Mineradio** 快捷方式。

> 旧版用户（v1.0.10 及更早）：先卸载旧版 → 重启电脑 → 再装新版。直接覆盖会出问题。

---

## 三、第三步：打开听歌

双击桌面 **Mineradio** 图标。你能看到：
- **粒子视觉**（电影镜头级特效，跟随音乐节奏）
- **3D 歌单架**（右键唤出）
- **天气电台**（界面融合天气信息）

登录网易云或 QQ 音乐后就能搜歌、听你歌单里的歌。核心功能：

| 功能 | 操作 |
|------|------|
| 歌曲搜索 | 点搜索框，输入歌名/歌手 |
| 歌单浏览 | 右键唤出 3D 歌单架 |
| 歌词舞台 | 自动显示，电影镜头感 |
| 播客 | 网易云播客也能听 |

## 四、⚠️ 避坑总结

| 坑 | 现象 | 解法 |
|---|---|---|
| 下载了 Source code | 解压了不知道咋用 | 只下载 `*-Setup.exe` |
| SmartScreen 拦了 | 点"更多信息"→"仍要运行" | 正常提示，非病毒 |
| 旧版覆盖安装 | 功能异常 | 先卸载旧版再装 |
| 杀毒报警 | 下载被删 | 重新下载，还报警截图反馈 |
