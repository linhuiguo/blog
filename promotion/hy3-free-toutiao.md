# 路线一：最简单——用 Hermes Agent 直接白嫖（无需自己搞 API）

适合人群：纯小白，不想碰密钥、不想配环境的人。

Hermes Agent 是一个桌面 AI 助手，它内置了腾讯 Hy3 免费选项，装好就能用。

## 步骤 1：注册并登录 Hermes Agent

1. 打开 Hermes Agent 官网（搜索 "Hermes Agent" 即可找到）
2. 点击右上角"登录"
3. 支持 **谷歌账号一键登录**，也可以用邮箱注册
4. 登录成功后，进入控制台页面

## 步骤 2：下载电脑版客户端

1. 在官网找到"下载"页面
2. 点 **下载 Windows 版本**（Mac / Linux 也有对应版本）
3. 浏览器提示"是否允许下载"，点"允许"
4. 等待下载完成（文件不大，几分钟）

## 步骤 3：安装并连接

1. 双击下载好的安装文件
2. 弹出安装界面，直接点"**安装**"按钮
3. 等进度条走完（需要一会儿，耐心等）
4. 安装完成，点"完成"关闭安装窗口
5. 打开 Hermes Agent 客户端，它会提示"连接到 Hermes Agent"
6. 因为你刚才已经在网页端登录过，直接点"**连接**"按钮
7. 它会跳转确认，再点一次"连接"就成功了

## 步骤 4：切换模型为腾讯 Hy3

1. 进客户端主界面，点左下角或右上角的**齿轮（设置）**图标
2. 先把语言改成**中文**（默认是英文，找到 Language 改成 中文）
3. 点"**模型**"这一项
4. 你会看到默认可能已经是"腾讯 Hy3 免费"
   - 如果不是：点模型下拉框 → 往外往下拉 → 找到 **Hy3**（大概在列表倒数第二个）→ 点它
5. 点"**应用**"保存

## 步骤 5：开聊！

在输入框打个"你好，你是什么大模型"，它会回"我是 Hermes Agent，底层运行的模型是腾讯 Hy3 免费大模型"。

第一次会稍慢（冷启动），第二、三次就快了。你可以让它写 HTML、搜资料、处理文件、生成图片、跑自动化任务、设定时任务。

> 小贴士：视频里博主让它写了个"X超哥博客单页"放到桌面，效果不错。你也可以试试让它帮你写个落地页或工具脚本。

# 路线二：最通用——用 OpenRouter 拿免费 API，配到任意工具

适合人群：想在自己常用的编程工具（Codex、Kilo Code、Cline、OpenCode、Claude Code）或聊天客户端（Cherry Studio）里用 Hy3 的人。

核心思路：OpenRouter 是一个"大模型超市"，它把腾讯 Hy3 免费档接进来了。你在 OpenRouter 申请一个 API Key，然后把这个 Key 填到任何支持 OpenAI 格式接口的工具里，就能用 Hy3。

## 关键信息（照抄）

- **API 端点（Base URL）**：`https://openrouter.ai/api/v1`
- **模型 ID**：`tencent/hy3:free`
- **API Key**：在 OpenRouter 后台自己创建（下面教）
- **接口格式**：标准 OpenAI 兼容，所有支持 OpenAI 接口的工具都能用

## 步骤 1：注册 OpenRouter 并创建 API Key

1. 打开 OpenRouter 官网（搜索 "OpenRouter"）
2. 点"登录 / Sign In"，用 **GitHub 一键登录**最方便（没有 GitHub 就注册邮箱）
3. 登录后，点右上角头像 → **Settings（设置）** → **API Keys**
4. 点"**Create**"（创建密钥）
5. 给密钥起个名字（随便填，比如 `my-hy3`）
6. 过期时间：自己用选 **Never（永不过期）**；临时测试可以选 1 小时或 1 天
7. 点 Create 后，**会显示一串 `sk-or-...` 开头的密钥**——**立刻复制保存**，它只显示一次！
   ⚠️ 密钥等同于你的账号密码，别发到公开地方

## 步骤 2：配到 Cherry Studio（聊天客户端，验证用）

Cherry Studio 是个桌面 AI 聊天软件，先拿它验证 Key 能用。

1. 打开 Cherry Studio，点"设置"
2. 往下拉找到"**模型配置**"，搜索框里搜 **OpenRouter**
3. 它会带出 OpenRouter 的配置项，API 地址已经填好了（`https://openrouter.ai/api/v1`）
4. 把刚才复制的密钥粘贴到"API 密钥"栏
5. 点"检测"——显示连接成功
6. 点"**获取模型列表**"，等它拉取
7. 在搜索框搜 `hy`，会看到 **Hy3 Free（标注免费）**，点"+"添加
8. 返回首页，点顶部模型下拉框，选 **Hy3 Free**
9. 发一句"你好"——秒回，说明通了

## 步骤 3：配到 Kilo Code（VS Code 里写代码）

Kilo Code 是 VS Code 的 AI 编程插件，在 OpenRouter 的 agent 使用排名里是第一。

1. 打开 VS Code，进"扩展"商店
2. 搜 **Kilo Code**，安装并登录账号
3. 登录后，在模型选择里能看到一个标着"**免费**"的大按钮——那就是腾讯 Hy3
4. 选它，就能在 VS Code 里免费用 Hy3 写代码
5. 测试：切到 Code 模式，输入"帮我生成一个坦克大战网页游戏"，回车
   - 它第一次生成可能不完整，补一句"自己测一下，保证能正常玩"，让它自我修复，等几分钟就能跑

> 实测：Hy3 写坦克大战能勉强跑通，但和高级模型比还有差距。它更适合**日常任务和一般开发**，复杂核心功能建议用更强的模型。性价比摆在那，免费还要啥自行车。

## 步骤 4：配到 Codex（OpenAI 的命令行工具）

Codex 也是 OpenAI 兼容接口，配置方式一样——给环境变量或配置文件填 Base URL + Key + 模型名。

在终端里设置环境变量（Windows PowerShell）：

```powershell
$env:OPENAI_BASE_URL = "https://openrouter.ai/api/v1"
$env:OPENAI_API_KEY = "sk-or-你的密钥"
$env:OPENAI_MODEL = "tencent/hy3:free"
```

设好之后，Codex 就会走 OpenRouter 的 Hy3 免费档。

> 说明：不同版本的 Codex 读的环境变量名可能略有差异（有的叫 `CODEX_BASE_URL` 之类），但逻辑都一样——把"接口地址、密钥、模型名"这三样填进去即可。

## 步骤 5：配到 OpenClaw 等同类工具

所有支持"自定义 OpenAI 兼容接口"的工具（Claude Code、Cline、OpenCode、OpenClaw 等）配置都长一个样：

| 配置项 | 填什么 |
|--------|--------|
| Base URL / API 地址 | `https://openrouter.ai/api/v1` |
| API Key / 密钥 | 你刚创建的 `sk-or-...` |
| Model / 模型名 | `tencent/hy3:free` |

找到工具里的"设置 → 模型 → 添加自定义 / 自定义端点"，把上面三行填进去，保存，选 Hy3 就能用。

# 常见问题

**Q：Hy3 真免费吗？**
A：在 OpenRouter 里 `tencent/hy3:free` 标注免费（价格 0），Hermes Agent 里也有内置免费档。但免费额度**有时间限制**（视频提到约两周），过期后可能收费或换档，趁早用。

**Q：第一次很慢正常吗？**
A：正常，冷启动慢，第二三次就快了。

**Q：能用来干啥？**
A：写代码、做网页、处理 Excel / PPT、搜资料、写文案、自动化任务、定时任务都行。视频里演示了 101 条销售数据建模出 30 页 PPT、汇总三地区数据生成 5000+ 单元格 Excel。

**Q：和 GPT 比怎么样？**
A：参数规模小，复杂任务有差距，但免费、日常够用、性价比高。

**Q：Codex / Kilo Code / Cline 选哪个？**
A：纯聊天验证用 Cherry Studio；VS Code 里写代码用 Kilo Code 或 Cline；命令行爱用 Codex。底层都是同一个 Hy3，只是前端不同。
