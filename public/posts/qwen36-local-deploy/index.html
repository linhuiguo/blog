前言#

最近 Qwen3.6 越狱版在社区火了，35B 参数的模型在 8G 显存上就能跑。这篇教程带你从零开始在 Windows 上部署。

环境要求#

  • Windows 10/11
  • NVIDIA GPU（8G+ 显存推荐）
  • llama.cpp 最新版

安装步骤#

1. 下载 llama.cpp#

git clone https://github.com/ggerganov/llama.cpp
cd llama.cpp
cmake -B build
cmake --build build --config Release

2. 下载模型#

从 HuggingFace 下载 GGUF 格式的模型文件。

3. 运行推理#

./build/bin/llama-cli -m qwen3.6-35b.gguf -ngl 99 -c 8192

性能表现#

在 RTX 3070 上实测，token 生成速度可达 15 tokens/s，完全可用。

总结#

本地部署大模型的门槛越来越低，8G 显存就能跑 35B 参数的模型，这对于个人开发者来说是个好消息。
