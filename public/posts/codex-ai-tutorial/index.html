前言#

Codex 是 OpenAI 推出的 AI 助手工具，功能强大，支持对话、代码生成、文件处理等多种场景。最重要的是，它有免费额度，用完后会定期重置，适合日常使用。

本文从下载安装到使用技巧，手把手带你走一遍全流程。

一、下载安装#

1.1 下载#

访问 OpenAI 官网下载 Codex 客户端：

  • 官网地址：https://codex.openai.com

选择对应你操作系统的版本（Windows / macOS / Linux），点击下载即可。

1.2 安装#

下载完成后，双击安装包运行。安装过程非常简单：

  • Windows：全程自动安装，无需手动选择路径。安装后程序默认放在 C:\Program Files\WindowsApps\OpenAI.Codex_...\app\ 目录下
  • macOS：拖动到 Applications 文件夹即可
  • Linux：按照提示完成安装

安装完成后，桌面会出现 Codex 的图标。

1.3 打开和登录#

打开 Codex 后，会要求你登录 OpenAI 账号。

重要提示：如果你在国内使用，务必先打开全局代理（TUN 模式），否则无法正常登录和使用。

登录方式支持：

  • OpenAI 账号密码登录
  • Google 账号授权登录
  • Apple ID 登录

登录成功后即可进入主界面。

1.4 卸载#

如果需要卸载 Codex：

Windows：开始 → 设置 → 应用 → 安装的应用 → 搜索 Codex → 卸载

macOS：打开 Finder → 应用程序 → 找到 Codex → 右键移到废纸篓

二、使用初体验#

2.1 免费额度#

Codex 提供免费额度，用完后会定期重置。这意味着你可以长期免费使用基本功能，不需要付费订阅。

2.2 选择模型和推理强度#

在对话框右下角，你可以选择：

  • 模型版本：选择不同的 AI 模型
  • 推理强度：强度越高，消耗 Token 越多，但结果越准确

对于日常使用，中等强度就够了。复杂任务可以调高强度。

2.3 权限管理#

左下角可以配置 Codex 的权限级别，这是使用前必须了解的重要设置：

权限级别 说明 适用场景
低权限 每次操作都会询问用户确认 新手入门、处理敏感文件
中权限 偶尔询问用户确认 日常使用（推荐）
高权限 AI 具备增删改查能力，不会询问用户 高级用户、自动化任务

建议：刚开始使用时选择中权限，熟悉后再根据需要调整。

2.4 创建内容#

点击左下角的加号，可以快速创建：

  • PPT 演示文稿：输入主题，AI 自动生成幻灯片
  • 文章：输入要求，AI 生成完整文章
  • 图片：输入描述，AI 生成图片
  • 文件：创建各类文档

2.5 计划模式和目标模式#

这两个模式是 Codex 的核心功能：

计划模式：开启后，AI 会先提供方案，由你决定是否执行。适合需要审核的复杂任务。

目标模式：开启后，AI 会一直执行提示词，直到完成为止。适合明确的自动化任务。

2.6 插件功能#

Codex 支持多种插件扩展功能：

浏览器插件：让 Codex 以浏览器视角查看网址，可以分析网页内容、提取信息。

PDF 插件：让 Codex 解析整理 PDF 文件，可以总结、提取关键信息、翻译等。

在左侧插件栏可以看到更多可用的插件和 Skill，根据需要启用即可。

2.7 自动化工具#

Codex 内置自动化工具，可以创建定时任务：

  • 定时发送邮件
  • 定时检查文件更新
  • 定时执行脚本
  • 自定义自动化流程

这让 Codex 可以成为你的私人助理，7×24 小时自动工作。

2.8 项目管理#

对话框下侧可以选择项目。选择项目后，所有对话内容都会以该项目展开，方便管理和回顾。

三、使用技巧#

3.1 提示词写法#

好的提示词 = 好的结果。建议：

  • 明确目标：告诉 AI 你想要什么，而不是怎么做
  • 提供上下文：背景信息越充分，结果越准确
  • 分步骤：复杂任务拆分成小步骤，逐步完成
  • 指定格式：明确输出格式（Markdown、表格、代码等）

3.2 善用插件#

  • 写代码时开启浏览器插件，可以实时查看文档
  • 处理 PDF 时开启 PDF 插件，可以自动提取内容
  • 需要自动化时配置定时任务，让 AI 替你干活

3.3 合理使用权限#

  • 处理重要文件时用低权限，防止误操作
  • 日常使用中权限，平衡效率和安全
  • 自动化任务用高权限，但要确保任务明确

四、常见问题#

Q: Codex 加载不出来怎么办？#

A: 确保全局代理已开启（TUN 模式），然后重启应用。

Q: 登录一直转圈怎么办？#

A: 检查网络连接，确保代理正常工作。如果还不行，尝试退出重新登录。

Q: 免费额度用完了怎么办？#

A: 等待定期重置即可，不需要付费。重置周期通常为几天。

Q: 如何提高生成质量？#

A: 提高推理强度，提供更详细的提示词，使用合适的插件。

总结#

免费使用 有免费额度，定期重置
