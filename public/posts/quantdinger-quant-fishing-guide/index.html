闲电脑挂机赚零花！量化钓鱼法，80%把握才出手，安全躺赚（保姆级）#

家里一台闲置电脑，不玩游戏、不剪辑、不办公，放着也是放着。用它跑量化程序，每天挂机几个小时，当副业赚点零花。

今天讲的不是高风险赌博，而是量化钓鱼法——核心就一个原则：80%把握才出手。宁可错过，绝不接飞刀。

⚠️ 先说清#

  1. 这是副业，不是主业：一个月赚几百到几千，图个零花，别指望暴富。
  2. 量化钓鱼法 = 提高触发门槛：宁可少交易，不接烂单。风控比赚钱重要。
  3. 需要一台闲置电脑：24 小时挂着，不用人管，断电了自动重启就行。
  4. 开源免费：用 QuantDinger（GitHub 9472 星，Apache-2.0 协议），Python 写的，谁都能装。

〇、准备清单#

  • 一台闲置电脑（Win10/Win11 都行，能 24 小时开）
  • Python 3.10+
  • 一个交易所测试网账号（Binance testnet，免费，不用真钱）

一、第一步：量化钓鱼法是什么#

量化钓鱼法有三个原则：

  1. 提高触发门槛：普通量化看到 30% 胜率就出手，钓鱼法必须 80% 以上才出手
  2. 宁可错过，绝不接飞刀：行情不好就空仓，空仓也是一种策略
  3. 固定止损：每笔交易最大亏损不超过本金的 2%

一句话总结：像钓鱼一样，耐心等，看准了再下钩。

二、第二步：安装量化程序#

git clone https://github.com/brokermr810/QuantDinger.git
cd QuantDinger
pip install -r requirements.txt

⚠️ 没装 Git？去 git-scm.com 下载。 ⚠️ pip 慢？加 -i https://pypi.tuna.tsinghua.edu.cn/simple

三、第三步：配置测试网（不用真钱）#

编辑 config.json：

{
  "exchange": "binance",
  "api_key": "你的测试网API Key",
  "api_secret": "你的测试网API Secret",
  "testnet": true,
  "trading_mode": "conservative",
  "min_win_rate": 0.8
}

⚠️ Binance 测试网 Key 怎么申请：

  1. 搜索 “Binance testnet”，进测试网网站
  2. 注册账号，进 Settings → API Management → 创建 Key
  3. 复制 Key 和 Secret 填进去
  4. testnet: true 就是模拟盘，亏了也不心疼

四、第四步：设置量化钓鱼参数#

在策略文件里设置：

参数 钓鱼法推荐值 说明
最低胜率 80% 低于 80% 不交易
单笔止损 2% 亏了不超过本金 2% 就止
单笔止盈 5% 赚 5% 就走，不贪
最大持仓 30% 永远留 70% 现金，不怕踏空
每日最大亏损 3% 一天亏 3% 就关机，明天再战

五、第五步：跑回测验证#

先回测一个月数据，看看策略在不加任何本金的情况下赚不赚钱：

python backtest.py --symbol BTCUSDT --strategy conservative --start 2025-06-01 --end 2025-07-01

如果回测结果是正收益 + 最大回撤小于 5%，说明策略过关，可以上测试网跑。

六、第六步：挂机运行#

python live_trading.py

程序会 24 小时自动运行，看准信号才出手，平时空仓等待。你不需要管，电脑自己跑。

⚠️ 避坑总结#

坑 现象 解法
连续亏损 几天都亏钱 触发每日最大亏损止损，关机冷静
回测好实盘差 回测赚实盘亏 加 testnet 多跑一周模拟
API Key 申请失败 测试网进不去 用 CC 代理翻墙申请
电脑断电 程序停止 用 Windows 任务计划设断电恢复自启动
贪心想赚更多 调高仓位想赚大钱 别动仓位，纪律比贪婪重要

一句话提醒#

量化钓鱼法的核心不是赚多少，而是少亏。 一个月赚 10%，比一个月亏 5% 然后想回本就强。
