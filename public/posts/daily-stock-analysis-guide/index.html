股市钓鱼，不拼运气！用 AI 科学选股，每日自动推送决策（保姆级）#

炒股拼运气是赌博，用 AI 科学选股才是钓鱼。

一个 56670 星的开源项目——daily_stock_analysis，用 LLM（大语言模型）驱动多市场股票智能分析，每日定时运行，实时新闻 + 决策看板 + 自动推送，零成本。

核心方法：股市钓鱼法——不追热点、不追涨停板，只抓 80% 以上把握的机会，宁可错过，绝不接飞刀。

⚠️ 先说清#

  1. 这是辅助工具，不是稳赚：AI 给你的是分析和建议，最终决策还是你来。
  2. 不需要交易所账号：纯分析工具，不牵涉资金，看完再决定是否动手。
  3. 零成本运行：用免费的 LLM API（如日日新 SenseNova 免费额度），不用花钱。
  4. 开源免费：Apache-2.0 协议，GitHub 56670 星，作者是大佬 ZhuLinsen。

〇、准备清单#

  • 一台电脑（Win10/Win11 都行，不需要高配）
  • Python 3.10+
  • 一个 LLM API Key（免费额度：SenseNova 日日新 1500 次/5 小时，或 OpenClaw 免费通道）

一、第一步：什么是股市钓鱼法#

炒股最大的坑就是追涨杀跌——涨了才买，跌了就卖。结果是赚小钱亏大钱。

股市钓鱼法有四个原则：

  1. 不追热点：热点已经起来的时候，鱼已经走了
  2. 只抓高把握：80% 胜率以下不出手，宁可空仓
  3. 定时分析：每天定时跑 AI 分析，不是盯着盘面
  4. 自动推送：手机收到决策看板，不打开任何行情软件

一句话：钓鱼要耐心等，看准了再下钩。

二、第二步：安装项目#

git clone https://github.com/ZhuLinsen/daily_stock_analysis.git
cd daily_stock_analysis
pip install -r requirements.txt

⚠️ 没装 Git？去 git-scm.com 下载，一路下一步。 ⚠️ pip 慢？加 -i https://pypi.tuna.tsinghua.edu.cn/simple

三、第三步：配置免费 LLM#

项目支持多种 LLM，最省钱的是日日新 SenseNova（免费额度充足）：

编辑配置文件，添加：

{
  "llm_provider": "sensenova",
  "api_key": "你的日日新 API Key",
  "api_base": "https://token.sensenova.cn/v1",
  "model": "sensenova-6.7-flash-lite",
  "analysis_frequency": "daily",
  "push_channel": "console"
}

⚠️ 日日新 API Key 在 platform.sensenova.cn 免费注册就能拿。

四、第四步：设置分析参数#

# 核心参数
ANALYSIS_MARKETS = ["A股", "美股", "港股"]
WATCH_SYMBOLS = ["贵州茅台", "宁德时代", "比亚迪", "腾讯控股"]  # 自己改
DECISION_RULES = {
    "min_confidence": 0.8,      # 80%把握才出手
    "max_daily_loss": 0.03,     # 单日最大亏损3%就停手
    "position_limit": 0.3,       # 仓位不超过30%
}

五、第五步：每日自动运行#

python main.py

程序会：

  1. 拉取所有关注股票的最新行情
  2. 抓取实时财经新闻
  3. 用 LLM 综合分析
  4. 生成决策看板
  5. 自动推送结果（控制台/微信/邮件）

每天早上 9 点前跑一次，开盘前看完就能决策。

六、它能干嘛#

能力 说明
多市场覆盖 A股、美股、港股同时分析
实时新闻 抓取财经新闻，解读影响
LLM 深度分析 大模型给基本面+技术面综合评分
决策看板 一目了然的买/卖/观望建议
自动推送 结果推送到微信/邮件/控制台

⚠️ 避坑总结#

坑 现象 解法
追热点 看到涨停才想买入 钓鱼法：80%把握才出手，追的热点直接无视
频繁交易 一天买卖好几轮 降低交易频率，每天只看一次决策看板
满仓梭哈 全部资金投入一只股 仓位上限30%，永远留70%现金
API Key 额度不够 免费额度用完了 换 SenseNova 免费额度（1500次/5小时）
不会改代码 看不懂 Python 直接改配置文件里的 WATCH_SYMBOLS 列表就行

一句话提醒#

股市钓鱼法的本质：不是赚得快，而是亏得少。 每天稳定赚一点，比一天赚10%然后想回本就强。
