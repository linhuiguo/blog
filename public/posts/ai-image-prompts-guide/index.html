前言#

你知道这些真人感十足的图片是如何生成的吗？

只需要一个简单的文生图指令，就可以做出一张这样的图片吗？

并非如此。好的图片不仅仅依赖模型，提示词也同样重要。

换句话说：好的图片 = 厉害的模型 + 厉害的提示词。

那么如何写出高质量的提示词呢？下面这个开源项目，应该可以帮到你。

项目介绍#

今天分享一个非常牛的开源项目，star 达到了 2.8k。

你可以在 GitHub 搜索 gpt4o-image-prompts，就能找到这个项目。

项目地址：https://github.com/AlenLiu99/gpt4o-image-prompts

第一步：浏览案例#

在项目的 README 页面，你可以找到示例图片以及示例提示词。

提示词技巧总结#

通过分析这个项目的提示词，可以总结出几个关键技巧：

1. 分层描述#

好的提示词通常分为多个层次：

层次 内容 示例
叙事层 整体风格和场景 冬季生活方式/旅行摄影
纹理层 皮肤、服装、环境的材质细节 苍白的瓷肌、亮面尼龙质感
相机层 镜头参数、对焦、光线 索尼 A7R V，35mm f/11

2. 具体参数#

提示词中要包含具体的参数，而不是模糊的描述：

  • ❌ “一个女孩” → ✅ “一位 19-25 岁的年轻女性，拥有白皙的肌肤，留着齐肩黑发”
  • ❌ “好看的背景” → ✅ “白雪皑皑的阿尔卑斯山脉、滑雪道、松树和湛蓝的天空”
  • ❌ “拍得好” → ✅ “索尼 A7R V，35mm 镜头，1/1000 秒，ISO 100”

3. 相机参数#

真实的相机参数能让图片更真实：

索尼 A7R V，35mm f/11，1/1000 秒，ISO 100
景深大（f/11）——无模糊
高对比度

4. 避免侵权#

如果你要生成图片用于商业用途，建议：

  1. 参考案例中的提示词结构
  2. 修改具体描述（人物、场景、服装等）
  3. 不要直接复制别人的提示词生成一模一样的图片

总结#

要点 说明
好的图片 厉害的模型 + 厉害的提示词
提示词结构 分层描述：叙事层 + 纹理层 + 相机层
关键参数 具体的人物描述、环境细节、相机参数
工具推荐 豆包、Gemini、Flux 等 AI 工具
开源项目 gpt4o-image-prompts（2.8k star）

总的来说，AI 提示词在文生图领域非常重要，提示词描述的细节越多，生成的图片就越精细。

如果你对文生图提示词的用法还不熟悉，可以仿照案例中的用法，稍作改造一下，就可以生成自己喜欢的图片，从而避免侵权的困扰。
