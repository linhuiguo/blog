白嫖 AI 生图！免费无限次，文生图图生图都行，小白也能用#

市面上 AI 生图工具要么收费，要么排队。今天分享一个免费、无限次、不用排队的 AI 生图 API Agnes Image 2.1 Flash——文生图、图生图、改图都支持，生成一张图只要几秒钟，目前完全免费。 本文从注册开始，每一步都给小白看。

⚠️ 先说清#

  1. 目前免费，以后未知：Agnes Image 2.1 Flash 目前定价 $0/张，趁现在免费赶紧用。
  2. 需要一点点命令行：生图只需要一行命令，我会教你复制粘贴就行。
  3. 不需要显卡：云端算力，你本地只需要能联网。

〇、准备清单#

  • 一台能上网的电脑（Win10/Win11 都行）
  • 一个邮箱（注册用）

一、第一步：注册并拿到 API Key#

  1. 浏览器打开 https://agnes-ai.com/zh-Hans/docs/agnes-image-21-flash
  2. 点右上角 登录/注册，用邮箱注册。
  3. 登录后进入控制台，找到 API Key 管理，点 创建 Key。
  4. 复制生成的 sk-... 密钥，存到记事本里（只显示一次）。

二、第二步：一行命令生图#

打开命令提示符（开始菜单搜 cmd 回车），把下面命令复制进去，把 YOUR_KEY 换成你的 Key：

curl -s --ssl-no-revoke -X POST "https://apihub.agnes-ai.com/v1/images/generations" \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d "{\"model\":\"agnes-image-2.1-flash\",\"prompt\":\"一只卡通猫，白色背景，可爱\",\"size\":\"1K\",\"n\":1}"

⚠️ 注意加了 --ssl-no-revoke（Windows 证书吊销检查可能连不上境外服务器，这个参数跳过它）。

等几秒，会返回一个 JSON，里面有一个 url 字段，复制它在浏览器里打开就是生成的图片。

三、支持的模型与参数#

模型 用途 价格
agnes-image-2.1-flash 文生图、图生图，推荐 $0/张
agnes-image-2.0-flash 文生图，旧版，稳定 $0/张
agnes-video-v20 视频生成 免费

尺寸支持档位：1K、2K、3K、4K，也兼容 1024x768 这种精确尺寸。

四、⚠️ 避坑总结#

坑 现象 解法
curl 报错 返回空或 401 检查 Key 是否复制对了，Bearer 后面有空格
SSL 报错 连不上服务器 加 --ssl-no-revoke
中文 prompt 乱码 服务器报 500 把 prompt 写入 .json 文件，用 -d @file.json 调用
生成慢 等超过 30 秒 先换 1K 尺寸，最快
